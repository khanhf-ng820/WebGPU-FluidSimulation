# WebGPU-FluidSimulation
(An attempt of making) Fluid Dynamics Simulation on the GPU.

*(only for incompressible, Newtonian fluids)*

- [Real-Time Fluid Dynamics for Games](http://graphics.cs.cmu.edu/nsp/course/15-464/Fall09/papers/StamFluidforGames.pdf)
- [Fast Fluid Dynamics Simulation on the GPU](https://cg.informatik.uni-freiburg.de/intern/seminar/gridFluids_GPU_Gems.pdf)
